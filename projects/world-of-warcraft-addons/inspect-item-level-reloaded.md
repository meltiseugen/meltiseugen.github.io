[Inspect Item Level Reloaded](https://github.com/meltiseugen/WoW-Inspect-Item-Level-Reloaded) is a World of Warcraft addon for showing average item level in the inspect frame.

The addon also tracks character and inspect equipment slots so low item-level pieces can be highlighted visually.

## Project details

- Repository: [meltiseugen/WoW-Inspect-Item-Level-Reloaded](https://github.com/meltiseugen/WoW-Inspect-Item-Level-Reloaded)
- Language: Lua
- Target: World of Warcraft Retail interface `120000+`
- Saved data: `InspectItemLevelReloadedDB`

## Main features

- Shows average item level in the inspect UI.
- Supports configurable inspect text size.
- Can color the average item-level text.
- Can show low item-level indicators on inspect and character slots.
- Supports configurable red threshold for items below average.
- Includes icon anchor settings for the low item-level indicator.
- Includes a fast inspect mode that skips tooltip fallback for faster inspection.

## Why it is useful

The inspect frame normally requires checking gear slot by slot. This addon surfaces the average item level and weak equipment slots directly in the UI, making quick inspection easier during group content, gearing checks, or character review.
