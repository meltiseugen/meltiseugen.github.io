Use this section for personal projects, technical experiments, and write-ups that are centered on something you built.

## Good fits for this section

- Side projects
- Build logs
- Technical deep-dives
- Case studies
- Postmortems

## Add a new page

Create a new Markdown file inside `projects/` and give it front matter like this:

```md
---
title: Project Name
parent: Projects
---
```

If you want a ready-made starting point, copy `scaffolds/project.md`.
