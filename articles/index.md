Use this section for standalone writing that does not belong to a specific project or hobby.

## Good fits for this section

- Notes and essays
- Tutorials
- Career thoughts
- Tooling write-ups
- Short reference pages

## Add a new page

Create a new Markdown file inside `articles/` and use:

```md
---
title: Article Title
parent: Articles
---
```

If you want a starting point, copy `scaffolds/article.md`.
